# 🩺 Patient Blood Test Analysis Dashboard

## 📌 Project Overview

This project is an interactive **Patient Blood Test Analysis Dashboard** designed to summarize and analyze blood test data in a **clear and user-friendly way**.

The dashboard provides a high-level overview of patient statistics and helps identify patterns and insights related to:

* **Age groups**
* **Gender distribution**
* **Hemoglobin levels**
* **Relationships between key blood test indicators**

## 📊 Key KPIs

* **Total Patients Count**
* **Average Glucose Level**
* **Average Age**
* **Average Hemoglobin Level**

---

## 🎛️ Interactive Filters

To support deeper analysis, the dashboard includes **slicers** for:

* **Gender:** Male / Female
* **Hemoglobin Range:** Low / Normal
* **Age Group:** 20–30 / 31–40 / 41–50 / Above 50

These filters allow users to **explore specific segments** of the data dynamically.

---

## 🔍 Key Insights

* The dataset shows a **balanced distribution** between male and female patients
* Most patients fall within the **31–40 age group**
* The **average hemoglobin level** is slightly below the defined target
* There is a **positive trend between WBC and Platelets**, suggesting a potential correlation
